## Key Features

### 1. Search & Filter
- **Search**: Real-time search by teacher name
- **Status Filter**: Draft / Submitted / Approved / All
- **Year Filter**: Filter by rating period (e.g., 2024-2025)

### 2. Main Table
- **Expandable Rows**: Click chevron to view rating history
- **Quick Actions**: Add Rating and View buttons
- **Color-Coded Status**: Visual indicators for rating status
- **Position Display**: Shows teacher's current position

### 3. Add Rating Form
- **Dynamic KRA Loading**: Automatically loads all KRAs and objectives
- **Rating Dropdowns**: 1-5 scale with descriptive labels
- **Auto-Calculation**: Scores calculated automatically based on weights
- **Real-time Updates**: Total score updates as you rate
- **Validation**: Ensures all required fields are filled

### 4. View Details
- **Comprehensive Overview**: Shows all rating details
- **Score Breakdown**: KRA-by-KRA analysis
- **Export Ready**: Prepared for PDF export (future enhancement)

### 5. Responsive Design
- **Mobile-Friendly**: Works on all screen sizes
- **Consistent Theme**: Matches ISAT DMS green theme
- **Background Watermark**: ISAT logo watermark maintained

### Example 3: Total Score and Numerical Rating
- **All Objectives**: 14 objectives with various ratings (3, 4, 5, etc.)
- **Total Score**: Sum of all objective scores = 71.40
- **Numerical Rating**: Average of all 14 objective ratings = 3.86 (on 1-5 scale)

Note: The numerical rating represents the average performance rating across all objectives, while the total score represents the weighted sum of all scores.
